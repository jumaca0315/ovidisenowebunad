INTRODUCCIÓN AL DISEÑO DE SITIOS WEB

OBJETIVOS

El curso de Diseño de Sitios Web está dirigido a estudiantes que estén interesados en adquirir conocimientos sobre el diseño, desarrollo e implementación de sitios web. Da a conocer complementaria sobre el uso de sistemas de control de versiones como GIT y GIHUB para gestionar proyectos colaborativos de diseño web, HTML5 y CSS3 para creación de contenidos en formato web como páginas y sitios Web. El curso esta estructurado en 3 unidades didácticas:

Unidad 1. Introducción al Diseño Web.

Unidad 2. GIT Y GITHUB.

Unidad 3. HTML 5.

Unidad 4. CSS3.
